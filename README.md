# ksopyla-research-blog

Source repository for Krzysztof Sopyla's AI research blog, published at [ai.ksopyla.com](https://ai.ksopyla.com/).

The site is built with [Hugo](https://gohugo.io/) and uses the [Congo theme](https://jpanther.github.io/congo/docs/). Content includes long-form blog posts, working drafts, LinkedIn posts, and X threads focused on AI research, experiments, and paper analysis.
